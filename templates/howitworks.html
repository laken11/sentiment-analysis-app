How It Works

1

Data acquisition

Twitter and reddit which are both popular social media application where people engage in discussions and expressed there thoughts and emotions was scrap for data based on some know stressors and there identifiers, each stressor scrap was able to generate at least 1000 text data dataset.

2

Data processing

After the data for each stressor has been scrapped and saved as a csv file containing just a colum called text, each cel with in this dataset is preprocessed to remove unnecessary characters and further tokenized. Tokenization is a text preprocessing step in sentiment analysis that involves breaking down the text into individual words or tokens(datacamp.com). This is an essential step in analyzing text data as it helps to separate individual words from the raw text, making it easier to analyze and understand(towardsdatascience.com). Tokenization also helps to remove punctuation, whitespace, and other irrelevant characters from the text. By tokenizing sentences, we can assign sentiment values to each word based on a sentiment lexicon, and then aggregate them to get the overall sentiment of the sentence(link.springer.com)

3

Sentiment analysis

Sentiment analysis is the process of analysis text and discovering the negative and positivity of the text, this is then done inline with our aim, which is to identify emotional state in text and classify them into True possible, True negative, False positive and False negative.Each dataset is also subjected to this and then eventually categorized into these category. A text is analysis True positive if it shows high sense of negativity and contains our identified stressors in them, a text is analysis False positive if it expresses negativity but does not contain the identified stressor, a text is True negative if it contains the identified stressor but does not show any form of negativity in them and finally a text is False negative if it shows no sign of negativity at all and does not contain the identified stressor.

4

Model training and testing

These models identified are then used to train and test these models for accuracy and other measures. The dataset that have been subjected to stage one, two and three where dividend into train ad test data which is then used for training and testing. After the testing, the accuracy score were taken and the trained models recorded high accuracy sources

4

Web application

A web application is developed,to enable users to select the identified stressor, pasted the text and subject them to analysis

5

Model classification

After these models where trained and tested, it is then used for classification of any given text. User select the identified stressor and input the text on the webpage then user request for analysis and the text is analyzed, and then the result is displayed the user
